# AWS Diagram

A simple diagram showing the common relationships between the most popular AWS services.

![The diagram](diagram.svg)

## Usage

```bash
$ brew install graphviz node
$ npm install
$ npm run build
$ open diagram.png
```

## Development

```bash
$ npm run watch
```
